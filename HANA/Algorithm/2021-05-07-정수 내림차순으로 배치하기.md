[연습문제]

함수 solution은 정수 n을 매개변수로 입력받습니다. n의 각 자릿수를 큰것부터 작은 순으로 정렬한 새로운 정수를 리턴해주세요. 예를들어 n이 118372면 873211을 리턴하면 됩니다.

#### 제한사항

- n은 1이상 8000000000 이하인 자연수입니다.

#### 입출력 예

| N      | return |
| ------ | ------ |
| 118372 | 873211 |

**code**

```js
function solution(n) {
  return Number(
    (n + '')
      .split('')
      .sort((a, b) => b - a)
      .join('')
  );
}
```

**description**
숫자를 문자열 -> 배열로 바꾼후 sort로 큰수대로 정렬한 후 join으로 합쳐서 숫자로 리턴한다.
