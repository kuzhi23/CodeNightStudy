[Frontend] client side rendering vs server side rendering ?


```
client side rendering
- 새로운 방식
- 브라우저에서 동적으로 라우팅
- 첫 접속, 로딩이 느림 최적화 필요

server side rendering
- 기존의 방식
- 서버내에서 모든 요청을 수행하기 때문에 더 빠름
- 웹 페이지를 미리 채워놓고 렌더링
```
