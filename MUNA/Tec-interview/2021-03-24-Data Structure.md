[Data Structure] stack vs queue 비교해서 설명해 주세요


```
stack은 후입선출 queue는 선입선출

스택의 활용 예시
1. 웹 브라우저 뒤로가기
2. 역순 문자열 만들기
3. 실행 취소(undo)
4. 후위 표기법 계산
5. 수식의 괄호 연산

큐의 활용 예시
1. 우선순위가 같은 작업 예약(프린터 인쇄 대기열)
2. 은행 업무
3. 콜센터 고객 대기시간
4. 프로세스 관리
5. 너비우선 탐색
6. 캐시 구현
```
