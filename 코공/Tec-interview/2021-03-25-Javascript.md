[Javascript] Difference between “==” and “===” ?

```
JavaScript에서 ==를 사용하면 연산이 되기 전에 피연산자들을 먼저 비교할 수 있는 형태로 변환시킵니다.
예를 들면 문자열을 숫자와 비교할 때 JavaScript는 모든 문자열을 숫자로 변환합니다. 빈 문자열은 항상 0으로 변환됩니다.
 숫자 값이없는 문자열은 NaN (숫자가 아님)으로 변환되어 false를 반환합니다.


=== (Triple equals)는 JavaScript의 엄격한 같음 비교 연산자로, 비슷한 유형이 아닌 값에 대해 false를 반환합니다.
이 연산자는 동일성을 위해 유형 캐스팅을 수행합니다. ===를 사용하여 2를 "2"와 비교하면 거짓 값이 반환됩니다.
```
